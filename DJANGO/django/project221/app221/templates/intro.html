Встановлення та налаштування Django

### Крок 1: Створення віртуального оточення

Використовуйте команду: python -m venv venv

### Крок 2: Активація

Для Windows (PowerShell): .\venv\Scripts\Activate.ps1

### Крок 3: Встановлення фреймворка

Виконайте команду: pip install django

### Крок 4: Створення проекту

django-admin startproject project221
